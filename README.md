🌟 About Matrix Bot

Matrix Bot MD is a multi-device WhatsApp bot built using Baileys, designed for:

Efficient group management
Fun interactive commands
Automation & moderation

Fully customizable and lightweight for personal or group use.

✨ Features
👥 Group Tools
.tagall — Mention everyone
Admin-only restricted commands
Anti-link & anti-spam protection
Warn, kick, promote & demote members
Mute / Unmute chat

🎮 Fun & Utility
Tic-Tac-Toe game
Text-to-Speech (.tts)
Sticker creation (.sticker)
Media & file utilities
More features coming!

🛠 Built With
Node.js
Baileys WhatsApp Library
JavaScript
