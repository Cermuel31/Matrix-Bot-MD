🌟 About Matrix Bot

Matrix Bot MD is a multi-device WhatsApp bot built using Baileys, designed for:

Efficient group management
Fun interactive commands
Automation & moderation

Fully customizable and lightweight for personal or group use.

✨ Features
👥 Group Tools
.tagall — Mention everyone
Admin-only restricted commands
Anti-link & anti-spam protection
Warn, kick, promote & demote members
Mute / Unmute chat

🎮 Fun & Utility
Tic-Tac-Toe game
Text-to-Speech (.tts)
Sticker creation (.sticker)
Media & file utilities
More features coming!

🛠 Built With
Node.js
Baileys WhatsApp Library
JavaScript

## 📖 About

Matrix Bot is a powerful and lightweight WhatsApp automation bot designed to give admins full control over their groups with ease.

Built on the Baileys library, it connects directly to WhatsApp Web and supports multi-device functionality for a smooth and reliable experience.

With features like automation, moderation tools, and interactive commands, Matrix Bot helps you:

Manage large groups effortlessly
Automate repetitive tasks
Improve group engagement
Maintain order and security

The bot runs on a Node.js environment and supports both QR code and pairing code authentication, making setup fast and simple.

## 🛠️ Setup & Installation

### Prerequisites

- Node.js installed on your system
- Git installed (for cloning the repository)

### Step-by-Step Setup

1. **Clone the repository:**

    ```bash
    git clone https://github.com/cermuel31/Matrixbot-MD.git
    cd Matrixbot-MD
    ```

2. **Install the dependencies:**

    ```bash
    npm install
    ```

3. **Run the bot:**

    ```bash
    node index.js
    ```

4. **Connect Your WhatsApp**
     
    ```QR code will appear in your terminal
    WhatsApp → Linked Devices
    Scan the QR code to connect
    ✅ Your bot is now live and ready to use!
    ```
